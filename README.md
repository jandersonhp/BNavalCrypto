## # 🔐 Encriptador BNaval

Um encriptador e decriptador de textos simples, desenvolvido com **Python** e **Flask**.
Disponível online: [Acesse aqui](https://bnavalcrypto.onrender.com/)

## 🚀 Funcionalidades

- ✅ Encriptar textos.
- ✅ Decriptar textos.
- ✅ Interface web simples e intuitiva.
- ✅ Deploy online usando Render.
